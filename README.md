# Myanmar Unicode to Zawgyi Converter in OCaml

### Usage

- `Unicode-zawgyi-converter.uni2zg input-text` (_ From unicode to zawgyi _)
- `Unicode-zawgyi-converter.zg2uni input-text` (_ From zawgyi to unicode _)
